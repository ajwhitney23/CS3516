---DIRECTIONS FOR USE---

---COMPILE INSTRUCTIONS---
open terminal in the directory this readme is located,
which at a minimum must contain:
	-node0.c
    -node1.c
    -node2.c
    -node3.c
    -project3.c
    -project3.h
	-makefile
    -NodeConfiguration

Run 'make all' to build the application
Run 'make clean' to delete both the executable before rebuilding

---RUNNING JUST THE PROGRAM---
open terminal in the directory in which the p2 file is located and
run the following command.
Usage: ./project3
